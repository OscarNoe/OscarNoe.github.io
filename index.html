Radio de circulo

Es bien sabido que para obtener el radio de un circulo se utuliza la formula:

Π x radio ²


entonces para poder llevar acabo nuestro programa es necesario contar con variables tipo float, ¿por que? debido a que obtendremos decimales en nuestros resultados y para eso es dicho tipo de variable, ok?

Para llevar acabo la operacion debémos definir el valor de Π (pi) en una variable constante para asi multiplicarlo posteriormente:

float total, radio;


una vez que tengamos nuestras variables podemos definir la operacion como lo hago a continuacion, para dejarlo claro la variable radio es donde va el valor de el radio que es introducido:

total = 3.1416 * radio * radio;
Area de un triangulo

Para poder optener el area de un triangulo en esta ocacion usaremos la siguiente formula, donde todas las variables son de tipo int

c = a*b/2;


como puedes observar contamos con tres variables, C es nuestro resultado A equivale a el area y B a la base de nuestro triangulo.

realmente no es tan complejo como lo imaginabas, sin embargo es importante que tengas presente como formular operaciones.
